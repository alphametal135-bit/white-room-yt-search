# سيرفر بحث YouTube Playlists — دليل التشغيل السريع

سيرفر صغير ومستقل تمامًا، وظيفته الوحيدة: استقبال اسم مادة دراسية،
والبحث عن أنسب قائمة تشغيل (Playlist) تعليمية على يوتيوب وإرجاعها.

**ملاحظة مهمة:** هذا السيرفر لا علاقة له بنظام تسجيل الدخول أو حفظ
البيانات — كل ده لسه شغال زي ما هو بالكامل عبر Firebase. السيرفر ده
بيضيف خاصية واحدة بس: البحث التلقائي عن كورسات للمواد الجديدة.

---

## 1) الحصول على مفتاح YouTube Data API (مجاني)

1. روح على https://console.cloud.google.com
2. اعمل مشروع جديد (أو استخدم مشروع موجود)
3. من القائمة الجانبية: **APIs & Services** → **Library**
4. ابحث عن **YouTube Data API v3** وفعّله (Enable)
5. روح على **Credentials** → **Create Credentials** → **API Key**
6. انسخ المفتاح اللي هيظهرلك

> الحصة المجانية اليومية: 10,000 وحدة، وعملية البحث الواحدة بتاخد
> 100 وحدة تقريبًا، يعني حوالي 100 عملية بحث مجانية يوميًا. السيرفر
> فيه كاش بسيط (12 ساعة) عشان يقلل الطلبات المكررة لنفس المادة.

## 2) تثبيت وتشغيل السيرفر محليًا للتجربة

```bash
npm install
YOUTUBE_API_KEY=ضع_مفتاحك_هنا npm start
```

السيرفر هيشتغل على `http://localhost:4000` (أو أي PORT تحدده).

اختبار سريع:
```
http://localhost:4000/api/search-playlist?q=رياضيات
```

## 3) رفع السيرفر على استضافة (مثال: Render، Railway، Vercel، إلخ)

- ارفع `youtube-search-server.js` و `package-youtube-search.json` (أعد تسميته إلى `package.json`)
- اضبط متغير البيئة `YOUTUBE_API_KEY` في إعدادات الاستضافة (مش في الكود مباشرة، عشان الأمان)
- بعد النشر هيديك رابط زي `https://your-app-name.onrender.com`

## 4) ربط الموقع بالسيرفر

في ملف `index.html`، دور على السطر ده (قرب إعدادات Firebase):

```js
const YT_SEARCH_SERVER_URL = "https://YOUR-YOUTUBE-SEARCH-SERVER-URL";
```

غيّره برابط السيرفر بتاعك اللي ظهر بعد النشر، مثلاً:

```js
const YT_SEARCH_SERVER_URL = "https://white-room-yt-search.onrender.com";
```

ارفع الملف المعدّل على Netlify زي العادة، وخلاص.

## ملخص سلوك النظام بعد التفعيل

- لو الطالب كتب اسم مادة من القائمة الجاهزة (برمجة، إنجليزي، إلكترونيات، فيزياء/كيمياء)
  → بتظهر المصادر فورًا من غير أي بحث (أسرع وموفّرة لحصة الـ API)
- لو كتب أي اسم تاني (رياضيات، تاريخ، عربي... إلخ)
  → الموقع بيكلم السيرفر، يبحث عن أنسب Playlist، ويعرضها كاملة جوا
    المشغّل المدمج، مع تحكم بالصوت (تعلية/تخفيض)
